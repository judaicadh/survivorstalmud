About this project

Between 1948 and 1950, in the Displaced Persons camps of the American occupation zone in Germany, survivors of the Holocaust — the She'erit Hapletah, the "surviving remnant" — printed a complete edition of the Babylonian Talmud. Known as the Survivors' Talmud (Talmud She'erit Hapletah), it was the first Talmud printed on German soil after the war, produced with the support of the U.S. Army.

This project traces the people, presses, and places connected to that undertaking — where the volumes were printed, where they circulated, and the communities that carried them onward.

How the map works

The map is a static site with no server or database. The data lives in a Google Sheet. When the sheet is published to the web as CSV, the map reads it live on load — so editing the spreadsheet updates the map with no code changes. The basemap is CARTO's Positron tiles, drawn with Leaflet.

Tracing movement

Each row in the sheet is one stop in a book's life — where it was, who held it, and what happened. Rows that share a book_id are grouped into a single journey and connected in order, so the map shows a copy moving and changing hands over time. Give the stops a sequence (1, 2, 3…) to set the order.

Contributing data

Add a row to the connected Google Sheet with these columns (only latitude and longitude are required):

  • book_id — groups the stops of one physical copy (e.g. copy-1)
  • sequence — order of this stop in the journey (1, 2, 3…)
  • date — when the book was here or changed hands
  • location — place name
  • latitude, longitude — decimal coordinates (required)
  • owner — who held it at this stop
  • event — what happened (printed, gifted, sold, donated, inherited…)
  • description, source, url — optional detail, citation, link

A row with no book_id still appears as a single standalone point, so you can mix individual locations with traced journeys.
