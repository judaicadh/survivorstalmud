About this project

Between 1948 and 1950, in the Displaced Persons camps of the American occupation zone in Germany, survivors of the Holocaust — the She'erit Hapletah, the "surviving remnant" — printed a complete edition of the Babylonian Talmud. Known as the Survivors' Talmud (Talmud She'erit Hapletah), it was the first Talmud printed on German soil after the war, produced with the support of the U.S. Army.

This project traces the people, presses, and places connected to that undertaking — where the volumes were printed, where they circulated, and the communities that carried them onward.

How the map works

The map is a static site with no server or database. Locations are stored in a Google Sheet with latitude and longitude columns. When the sheet is published to the web as CSV, the map reads it live on load — so editing the spreadsheet updates the map with no code changes. The basemap is CARTO's Positron tiles, drawn with Leaflet.

Contributing data

Add a row to the connected Google Sheet with these columns:

  • title — name of the place or event
  • latitude, longitude — decimal coordinates (required)
  • date — optional
  • description — optional
  • source, url — optional citation and link
